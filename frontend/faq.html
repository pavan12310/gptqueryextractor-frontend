Getting Started
How do I install GPT Query Extractor?+
Click "Add to Chrome" on the Chrome Web Store page. Once installed, the extension icon appears in your toolbar. Open ChatGPT.com, click the extension icon, and you're ready to go — no configuration needed.
What ChatGPT plan do I need?+
The extension works best with ChatGPT Plus or Team plans that have web browsing enabled. The query interception captures search queries generated when ChatGPT uses its web search tool. Free ChatGPT accounts with limited web access will produce fewer captured queries.
Which browsers are supported?+
Currently Chrome and all Chromium-based browsers (Edge, Brave, Arc, Opera) are fully supported. Firefox support is on the roadmap for a future release.
How do I load my prompts?+
You can either paste prompts directly into the text area (one per line) or upload a .txt or .csv file using the upload zone. The extension supports hundreds of prompts per session.
Features & Usage
How does query interception work?+
The extension injects a script that intercepts ChatGPT's internal network requests at the browser level. When ChatGPT uses its web search tool in response to your prompt, it generates search queries before hitting the web. The extension captures these queries before they're sent.
What is the delay setting for?+
The delay (15–90 seconds) controls how long the extension waits between sending prompts. It serves two purposes: giving ChatGPT enough time to generate and complete its response, and mimicking human-like usage to avoid triggering rate limits. We recommend 30 seconds as a safe default.
What does "Auto New Chat" do?+
Auto New Chat automatically starts a fresh ChatGPT conversation every 20 prompts. This prevents context from accumulating across prompts (which can affect query generation) and avoids hitting per-session rate limits on ChatGPT.
What export formats are available?+
Free plan: JSON and CSV. Pro plan: JSON, CSV, plain text (one query per line), and clipboard copy for any individual query. All formats include the original prompt associated with each set of queries.
Privacy & Safety
Does the extension store or send my data anywhere?+
No. All data — your prompts, extracted queries, and session history — is stored locally in your browser's chrome.storage. Nothing is sent to our servers. The extension only communicates with chatgpt.com, exactly as a normal ChatGPT session would.
Will this get my ChatGPT account banned?+
The extension mimics normal user behavior with configurable delays. Using it responsibly (recommended 30s delay, reasonable session sizes) is very low risk. However, running extremely high volumes continuously could trigger OpenAI's automated systems. Use at your own discretion and review OpenAI's Terms of Service.
What permissions does the extension require?+
The extension requests: storage (to save session data locally), tabs (to open ChatGPT), notifications (optional, to alert when a session completes), and host access to chatgpt.com only. It does not request access to any other websites.
