// Questions

FAQ

How do I install the extension?+
Click "Add to Chrome — Free" on this page or visit the Chrome Web Store directly. Click "Add to Chrome", confirm the permissions, and the extension icon will appear in your toolbar. Open ChatGPT, click the extension icon, and you're ready to run your first prompt batch.
Does it work with any ChatGPT plan?+
The extension works best with ChatGPT Plus or Team plans that have web browsing enabled. The query interception specifically captures the search queries generated when ChatGPT uses its web search tool. A free ChatGPT account with limited web access will produce fewer captured queries.
Is the extension really completely free?+
Yes — completely free. There is no paid plan. Install from the Chrome Web Store, no account or credit card required. All features including bulk prompts, all export formats, session history, auto new chat, and pause/resume control are available at no cost.
Will it get my ChatGPT account banned?+
The extension simulates normal user behavior and includes configurable delays between prompts. The recommended 30-second delay mimics human usage patterns. However, running extremely high volumes continuously is at your own risk. The auto new chat feature helps stay within OpenAI's per-session limits.
Does it store or send my prompts anywhere?+
No. All data stays local in your browser's storage. The extension communicates only with chatgpt.com — your prompts and extracted queries never leave your machine except to reach OpenAI's servers as they normally would in any ChatGPT session.
How many prompts can I run in one session?+
The free tier supports up to 50 prompts per session. The Pro tier has no hard limit. In practice, session length depends on your ChatGPT plan's rate limits and the delay setting. With a 30s delay, 100 prompts takes roughly 50 minutes.
Does it work with GPT-4o and the latest models?+
Yes. The extension works with all ChatGPT model variants including GPT-4o and GPT-4 Turbo. Query interception is model-agnostic — it captures searches at the network level regardless of which model is active, as long as the model has web browsing enabled.
What browsers are supported?+
Currently Chrome and all Chromium-based browsers (Edge, Brave, Arc, etc.) are fully supported via the Chrome Web Store. Firefox support is on the roadmap.
Does it work on mobile?+
The extension requires a desktop browser — Chrome extensions are not supported on mobile Chrome or Safari on iOS/Android. For best results use it on a desktop or laptop running Chrome, Edge, or Brave.
