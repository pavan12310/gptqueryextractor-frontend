FAQ

Common Questions

What does GPT Query Extractor actually do?+
It intercepts the internal search queries that ChatGPT fires when processing prompts. When ChatGPT searches the web to answer a question, it generates specific search queries internally — this extension captures those in real time.
Does it store or send my prompts anywhere?+
No. All data stays local in your browser's storage. The extension communicates only with chatgpt.com — your prompts and extracted queries never leave your machine except to reach OpenAI's servers as they normally would.
How many prompts can I run in one session?+
The free tier supports up to 50 prompts per session. In practice, session length also depends on your ChatGPT plan's rate limits and the delay setting you choose.
Does it work with GPT-4o and the latest models?+
Yes. Query interception is model-agnostic — it captures searches at the network level regardless of which ChatGPT model is active, as long as web browsing is enabled.
What browsers are supported?+
Chrome and all Chromium-based browsers (Edge, Brave, Arc) are fully supported. Firefox support is on the roadmap.
